Métier de l'informatique

De nos jours, les métiers de l'informatique évoluent énormément avec l'apparition de nouveaux outils comme l'apprentissage automatique (ou Machine Learning ), le traitement du langage naturel ( NLP ), la vision par ordinateur , l' automatisation des processus ...
Ces différentes solutions permettent de faciliter grandement divers tâches, voir même de laisser un programme les faire à notre place.

Dans cette partie, nous allons vous présenter divers exemples d'intelligences artificielles et les domaines dans lequels on peut les utiliser sans nous attarder sur leurs fonctionnement. Nous aborderons cela dans la section pour en savoir plus.

Exemples d'intelligences artificielles:

  • ChatGPT

    Certainement l'intelligence artificielle la plus connue aujourd'hui, ChatGPT , pour Chat Generative Pre-trained Transformer, est un outil crée par OpenAI. Il existe aujourd'hui 2 version de ChatGPT: GPT-3.5 : nécessite un compte OpenAI et est gratuit, et GPT4, plus performant mais nécessite un abonnement payant.

  • Copilot

    Crée par Microsoft et OpenAI, Copilot est un outils de développement payant qui permet de générer et traduire du code dans divers langages de programmation.

  • Intellicode

    Intellicode est un outils d'auto-complétion gratuit crée par Microsoft. Il permet de détecter le contexte du code, les noms de variable, les fonctions ainsi que le type de code que vous écrivez pour améliorer les suggestions d'auto-complétion. De plus, Intellicode s'exécute sur votre ordinateur pour garantir que votre code reste privé.

  • Bard

    Bard est un outils similaire à ChatGPT crée par Google. Il est également capable de traduire du texte, générer du code...

  • Uizard.io

    Uizard.io Est un outils en ligne qui permet de créer des pages web et applications mobiles facilement. Il possède une intelligence artificielle qui permet de créer une page web à partir d'un dessin.

  • NetApp

    NetApp est outils de cybersécurité qui permet de détecter automatiquement la moindre anomalie pour éviter les attaques.
